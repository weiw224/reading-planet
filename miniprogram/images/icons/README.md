# Icon Assets

This directory contains TabBar icons for the WeChat Mini Program.

## Required Icons

### TabBar Icons (48x48px recommended)

**Home (首页):**
- home.png - Unselected state
- home-active.png - Selected state

**Practice (练习):**
- practice.png - Unselected state
- practice-active.png - Selected state

**Profile (我的):**
- profile.png - Unselected state
- profile-active.png - Selected state

## Design Guidelines

- Color: #999999 (unselected), #F5A623 (selected)
- Format: PNG with transparency
- Size: 48x48px or 81x81px (2x/3x for retina)
- Style: Simple, clean, consistent with overall design system
